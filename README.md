## 🛠️ Deployment

### Phase 1: The Core (Backend)
```bash
cd backend
pip install -r requirements.txt
# Update your GROQ_API_KEY in backend/.env
python3 -m uvicorn app.main:app --reload
```

### Phase 2: The Interface (Frontend)
```bash
cd frontend
npm install
npm run dev
```
